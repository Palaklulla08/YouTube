# YouTube
building YouTube clone using mern stack

Overview
This project is a YouTube clone built using the MERN stack (MongoDB, Express.js, React.js, Node.js). It mimics basic functionality of YouTube, including user authentication, video upload, video search, category filters, and video playback. The project is divided into frontend and backend folders.

  cd backend
npm install

command to run is npm run dev

for front end :
cd frontend
npm install
